# Telegram Casino 🎰

Полнофункциональное казино для Telegram с поддержкой криптовалюты TON, Telegram Stars и традиционных платежей.

## Особенности

### 🎮 6 Игр Казино
- **Блэкджек** - Классическая игра 21 с автоматическим дилером
- **Рулетка** - Ставки на числа, цвета, четные/нечетные
- **Краш** - Множитель растет, нужно вовремя забрать
- **Бомбер** - Поле с минами, избегайте взрывов
- **Кости** - Игра в кости с различными ставками
- **Слоты** - Слот-машина 3x3 с различными символами

### 💰 Система Платежей
- **TON** - Оплата криптовалютой TON
- **Telegram Stars** - Внутренняя валюта Telegram
- **Карты** - Традиционные платежи через Stripe

### 🌐 Мультиязычность
- Русский язык (полная поддержка)
- Английский язык (полная поддержка)
- Переключение языка в профиле

### ✨ Современный Дизайн
- Dark theme с neon акцентами (#00FF66, #FF007A)
- Mobile-first подход (Telegram Mini App)
- Анимации с Framer Motion
- Glassmorphism эффекты

## API Endpoints

### Пользователи
- `POST /api/users` - Создать/получить пользователя
- `GET /api/users/{telegram_id}` - Получить данные пользователя
- `PATCH /api/users/{telegram_id}` - Обновить пользователя

### Игры
- `POST /api/games/play` - Сыграть в игру
  ```json
  {
    "telegram_id": 123456789,
    "game_type": "blackjack",
    "bet_amount": 10.0,
    "game_params": {}
  }
  ```

### Платежи
- `POST /api/deposit` - Пополнить баланс
  ```json
  {
    "telegram_id": 123456789,
    "amount": 100.0,
    "payment_method": "ton"
  }
  ```

### История
- `GET /api/history/{telegram_id}` - История игр
- `GET /api/transactions/{telegram_id}` - История транзакций

## Игровая Механика

### Блэкджек
- Дилер автоматически играет по классическим правилам
- Blackjack (21 с двух карт) платит 1.5x
- Обычный выигрыш платит 1x

### Рулетка
- Числа 0-36
- Красные числа: 1,3,5,7,9,12,14,16,18,19,21,23,25,27,30,32,34,36
- Черные числа: 2,4,6,8,10,11,13,15,17,20,22,24,26,28,29,31,33,35
- Типы ставок:
  - Число (35x)
  - Цвет (1x)
  - Четное/Нечетное (1x)
  - Высокое/Низкое (1x)

### Краш
- Автоматический кешаут между 1.5x и точкой краша
- Краш-поинт генерируется случайно от 1.01x до 100x

### Бомбер
- Поле 5x5 с 5 бомбами
- Каждая открытая безопасная ячейка увеличивает множитель
- Попадание на бомбу = проигрыш

### Кости
- Два кубика (1-6)
- Ставки: Over 7, Under 7, Exact sum, Even, Odd

### Слоты
- Сетка 3x3
- Символы: 🍒🍋🍊🍇🔔💎7️⃣
- Выигрыш за совпадение символов в линиях
